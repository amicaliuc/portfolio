# amicaliuc.com

Portfolio of Anatolie Micaliuc — product designer for B2B / B2C.

## Stack

Static HTML / CSS / JS. No frameworks, no build step.

## Structure

```
/
├── Index.html          # Main page
├── vision.html         # Case study: Enterprise F&B Vision (GoParrot/Square)
├── checkout.html       # Case study: Checkout Redesign
├── royalty.html        # Case study: Royalty Fee Collection
├── styles.css          # Global styles
├── transitions.js      # Custom cursor
├── fonts/              # Self-hosted: Stratos, Editorial New
└── img/                # Images (WebP + PNG fallback)
```

## Deploy

Auto-deploys to [amicaliuc.com](https://amicaliuc.com) via Netlify on push to `main`.
